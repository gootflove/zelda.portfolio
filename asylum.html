Lunatic Asylum

GAME DEVELOPMENT

// PROJECT OVERVIEW

This project is a 3D horror-themed asylum environment developed in Unity, where players must navigate through an asylum to retrieve their missing medicine. Throughout the game, players will explore different sections of the asylum while overcoming various obstacles, hazards, and environmental challenges that block their path.

// LEARNING OBJECTIVES

- Do a tracking system for the player to collect items and progress through the game.

- Implement physics-based interactions, such as pushing objects.

// AREAS OF IMPROVEMENT

The entire game is so boring and empty without any enemies or puzzles to make the gameplay more engaging. I would like to add in some simple enemy AI and puzzle mechanics to make the game more fun to play.
