// PROJECT OVERVIEW

Paws and Print is a keychain making company where you can create custom keychains by uploading your own pet photos or select from the already designed options. The website allows users to easily design and order personalized keychains, making it a fun and creative experience for customers.

There is also a virtual claw machine game where users can win discounts and prizes, adding an element of gamification to the shopping experience.

There is a feed page where users can share photos of their keychains and interact with other pet lovers, fostering a sense of community around the brand.

// LEARNING OBJECTIVES

- Implement gamification elements, user experience design, and interactive features in a web application using Figma.

// AREAS OF IMPROVEMENT

The user interface could be improved with more intuitive navigation and enhanced visual design to create a more engaging user experience.
