Science Lab in Spaceship

ENVIRONMENT DESIGN 3D MODELING TEXTURING

// PROJECT OVERVIEW

This project is a 3D science lab environment set inside a spaceship. The environment features various scientific equipment, workstations, and interactive elements that allow players to engage with the space and conduct experiments.

// LEARNING OBJECTIVES

- Learn how to use modular assets.

- Implement VFX and particle systems in Unity to enhance the visual appeal of the environment.

// AREAS OF IMPROVEMENT

The environment could be enhanced with more dynamic lighting and improved audio design to create a more immersive experience.

// PROJECT INFO

MODULE 3D FOR REAL TIME
ENGINE Unity
TOOLS Unity, Maya, C#, Substance Painter
DURATION 4 weeks July 2025 – Aug 2025
COLLABORATION Solo
