Modern Home

ENVIRONMENT DESIGN GAME DEVELOPMENT

// PROJECT OVERVIEW

This is a modern residential design project that showcases a sleek and functional home with clean lines and well-thought-out spaces.

// LEARNING OBJECTIVES

- Learn how to import and utilize 3D asset packs from the Unity Asset Store to rapidly prototype and build detailed environments.

- Process lighting and post-processing techniques in Unity to create a specific mood and atmosphere for the scene.

- Use Probuilder Tool in Unity to build custom geometry for unique architectural features and level design elements.

// AREAS OF IMPROVEMENT

I could have improved the space in different rooms as some rooms felt a bit cramped and some were too large.

// PROJECT INFO

MODULE Spatial Theory and Level Design
ENGINE Unity
TOOLS Unity, Unity Asset Store
DURATION 3 weeks May 2025 – June 2025
