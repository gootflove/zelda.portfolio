// PROJECT OVERVIEW

This is a 10 second promotional video for a climbing chalk called Psyche Chalk.

The video is designed to be visually engaging and to convey the brand's message in a short amount of time.

// LEARNING OBJECTIVES

- Thumbnails of the Short's visual flow so that anyone can follow.

- Style & Art Direction References such as set design, mood, color, clothing, hair, etc.
