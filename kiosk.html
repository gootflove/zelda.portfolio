// PROJECT OVERVIEW

This ICT Kiosk is meant to help users navigate the ICT campus and its services. It provides information about the courses, its societies, and its events. Users can also access the Ngee Ann Polytechnic's exclusive merchandise.

There is also a mini quiz where you can find out how you usually are in a team and an art wall, adding elements of gamification to the shopping experience.

These features helps people from both the ICT and non-ICT community to learn more about the school and its culture, while also providing a fun and interactive experience.

// LEARNING OBJECTIVES

Design a user-friendly and interactive kiosk interface which is easy for users of all ages to navigate and access information.

// AREAS OF IMPROVEMENT

The user interface could be improved as the pages are very text-heavy and could be more visually engaging.
