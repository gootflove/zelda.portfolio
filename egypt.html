Egyptian Hideout

ENVIRONMENT DESIGN GAME DEVELOPMENT

// PROJECT OVERVIEW

This project is a 3D Egyptian hideout environment set on an island. The environment features various artifacts, hieroglyphs, and interactive elements that allow players to explore the mysteries of the pharaoh's final resting place.

// LEARNING OBJECTIVES

- Use Unity's Tree tool to create a realistic palm tree.

- Made a flag that waves using shader graph.

// AREAS OF IMPROVEMENT

The environment could have had more items and details to make it feel more lived in and immersive. I also would have liked to add more VFX, such as dust particles or water effects, to enhance the atmosphere of the environment.

// PROJECT INFO

MODULE 3D Environment
ENGINE Unity
TOOLS Unity
